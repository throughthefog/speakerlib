# mdisk (Music Disk)
## Basic Format
### Must have:
1. left.dfpwm
2. metadata.json
### For stereo the following file needs to be included:
3. right.dfpwm

## Metadata
Must include all the information the following example does:
```json
{
	"metaver": 1,
	"artist": "Lemon Demon",
	"album": "Dinosaurchestra",
	"song": "Fine",
	"year": 2006
}
```
